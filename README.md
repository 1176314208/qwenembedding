# 基于 QwenAudio 的音频向量嵌入

本项目基于 [QwenAudio](https://github.com/QwenLM/Qwen-Audio) 的音频编码器，用于将音频数据转换为高维向量表示（音频嵌入）。这些嵌入可以进一步用于任务，例如音频分类、检索、聚类、语音识别等。
使用前请先根据[QwenAudio](https://github.com/QwenLM/Qwen-Audio)指示安装
